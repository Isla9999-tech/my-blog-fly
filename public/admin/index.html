📖 如何发布文章

  1. 在「写文章」页面填写标题、分类、正文
  2. 点击「生成文章文件」预览效果
  3. 点击「下载文件」获取 .md 文件
  4. 把文件放到项目的 content/blog/zh-CN/ 目录
  5. 推送到 GitHub：git add . && git commit -m "新文章" && git push
  6. Cloudflare Pages 自动部署，1-2 分钟后生效

🖼️ 如何添加图片

  1. 把图片放到 public/img/ 目录
  2. 在文章中使用：![描述](/img/图片名.jpg)
  3. 也可以直接用网络图片链接

🔗 如何管理友链

编辑 content/link.zh-CN.json 文件，格式：

[{
  "class_name": "友情链接",
  "class_desc": "推荐的网站",
  "link_list": [
    {
      "name": "网站名",
      "link": "https://example.com",
      "avatar": "https://example.com/avatar.png",
      "desc": "网站描述"
    }
  ]
}]
